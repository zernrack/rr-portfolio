

# Software Engineer Portfolio — Single Page, Minimal & Modern

## Design Direction
Clean, minimal single-page portfolio with generous whitespace, smooth scroll navigation, and typography-driven hierarchy using DM Sans / Lora / IBM Plex Mono. Light/dark mode toggle using the provided oklch color tokens. Green primary accents used sparingly for emphasis.

## Color & Theme
Apply the full oklch color system provided (light + dark) with a toggle in the navbar. Fonts: DM Sans for body, Lora for headings/quotes, IBM Plex Mono for code/tech labels.

## Sections (top to bottom)

### 1. Navigation Bar (sticky)
- Minimal sticky header with name/logo on the left
- Anchor links: About, Skills, Experience, Projects, Blog, Contact
- Dark mode toggle button on the right
- Smooth scroll behavior

### 2. Hero Section
- Large headline with name and role ("Software Engineer")
- One-line tagline/bio
- Subtle CTA buttons: "View Projects" and "Get in Touch" (anchor links)
- Clean, spacious layout — no clutter

### 3. About Section
- Short personal bio paragraph (2-3 sentences, placeholder text)
- Optional small avatar/photo placeholder
- Minimalist card or just clean typography on background

### 4. Skills / Tech Stack
- Grid of tech icons/labels organized by category (Frontend, Backend, Tools, etc.)
- Pill/badge style using the accent and secondary colors
- Clean, scannable layout

### 5. Experience / Timeline
- Vertical timeline with company, role, date range, and brief description
- Alternating or left-aligned minimal cards
- 3-4 placeholder entries

### 6. Projects Section
- Card grid (2-3 columns on desktop, stacked on mobile)
- Each card: project title, short description, tech tags, links (GitHub / Live)
- Subtle hover elevation using the shadow tokens
- 4-6 placeholder projects

### 7. Blog / Writing Section
- Simple list or card layout for articles
- Title, date, short excerpt, "Read more" link
- 3-4 placeholder entries

### 8. Contact Section
- Simple contact form (name, email, message) — frontend only, no backend
- Social links row (GitHub, LinkedIn, Twitter/X, Email icons)
- Clean footer with copyright

## Interactions & Polish
- Smooth scroll between sections
- Subtle fade-in animations on scroll (CSS-based)
- Responsive design: mobile-first, works beautifully on all screen sizes
- All content is placeholder/editable — easy to customize later
